# 東光醫療器材 — 商品目錄網站

## 如何新增商品

### 步驟一：建立商品資料夾

在 `products/` 資料夾內，依照以下結構建立目錄：

```
products/
  大類名稱/
    小類名稱/
      商品名稱/
        images/
          1.jpg    ← 主圖（一定要叫 1.jpg）
          2.jpg    ← 其他圖片（選填）
        info.txt
```

**資料夾名稱就是網站上顯示的名稱**，不需要另外設定。

> 也可以只用兩層（大類/商品）或一層（商品），系統會自動判斷。

---

### 步驟二：建立 info.txt

在商品資料夾內建立 `info.txt`，複製以下範本並填入資料：

```
名稱: 你的商品名稱
售價: NT$3,500
品牌: 品牌名稱
說明: 商品說明文字，可以寫得詳細一點。

規格: 尺寸 | 45x40x90 cm
規格: 重量 | 12 kg
規格: 承重 | 100 kg

標籤: 保固一年
標籤: 可申請補助
```

**說明：**

| 欄位 | 說明 | 是否必填 |
|------|------|----------|
| `名稱` | 商品顯示名稱 | 否（預設用資料夾名） |
| `售價` | 例如 `NT$3,500` | 否（空白顯示「歡迎洽詢」） |
| `品牌` | 商品品牌 | 否 |
| `說明` | 商品描述 | 否 |
| `規格` | 格式：`項目 \| 數值`，可以有多行 | 否 |
| `標籤` | 小標籤，可以有多行 | 否 |

---

### 步驟三：上傳圖片

- 主圖請命名為 `1.jpg`，網站封面會使用這張
- 其他圖片命名為 `2.jpg`、`3.jpg`...，會顯示在詳細頁的縮圖列
- 支援格式：JPG、PNG、WebP

---

### 步驟四：推送到 GitHub

將變更上傳（推送）到 GitHub 的 `main` 分支後：

1. GitHub 會自動執行程式，重新整理商品索引
2. 約 1~2 分鐘後，網站就會出現新商品

> 如果只有 `info.txt` 缺少圖片，商品仍然會出現，只是顯示預設圖片。

---

## 如何修改商品

直接編輯 `info.txt` 或替換圖片，然後推送到 GitHub 即可。

## 如何刪除商品

刪除整個商品資料夾，然後推送到 GitHub。

---

## 本機預覽

用 VS Code 安裝「Live Server」擴充套件，右鍵點選 `index.html` → **Open with Live Server**。
